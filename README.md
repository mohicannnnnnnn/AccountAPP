本次实现的安卓 记账程序 主要功能如下:

1）具有用户注册和登录的基本功能，只管理自己数据无法查看 

2）能够记录个人的每一笔收入和支出信息；

3）能够管理收入、支出的历史记录，实现查询更改及删除功；

4）能够按星期、月年统计收入和支出数据，并输结果 ;


主要实现除了界面就是数据库，刚开始不是很熟悉，所以很多都请教了同学。
在开始设置一个单表，所以可以实现多用户注册登录，但是数据还是共享的哈哈哈~
